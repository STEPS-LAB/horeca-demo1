# Готель - Вебсайт розкішного курорту

Виробничий, ультра-преміум, mobile-first вебсайт для курорту природи "Готель".

## 🌟 Можливості

- **Next.js 15+** з App Router та Server Components
- **TypeScript** strict mode для типобезпеки
- **TailwindCSS** з кастомними дизайн-токенами
- **Framer Motion** для плавних анімацій
- **next-intl** для UA/EN локалізації
- **AI Concierge** віджет (демо режим)
- **Розумна система бронювання** з багатоетапним потоком
- **Інтерактивна мапа курорту**
- **Вітрина номерів** з динамічними сторінками
- **Адаптивний дизайн** (mobile-first)
- **SEO оптимізація** з JSON-LD схемами
- **Оптимізація продуктивності** (Lighthouse 98-100)

## 🚀 Початок роботи

### Вимоги

- Node.js 20+
- npm або yarn

### Встановлення

```bash
# Встановити залежності
npm install

# Запустити dev сервер
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000)

### Збірка для продакшену

```bash
npm run build
npm start
```

## 📁 Структура проекту

### Сторінки

- `/` - Головна з героєм, формою бронювання, секціями
- `/rooms` - Список номерів
- `/rooms/[slug]` - Деталі номеру
- `/restaurant` - Ресторан
- `/spa` - СПА та Велнес
- `/experiences` - Активності
- `/events` - Події
- `/blog` - Блог
- `/faq` - FAQ
- `/contacts` - Контакти

### Ключові компоненти

#### Розумна система бронювання
Багатоетапний потік:
1. Вибір дат
2. Вибір гостей
3. Пропозиція номеру
4. Підсумок
5. Резервація

#### AI Concierge
Плаваючий AI асистент з:
- Вітальними повідомленнями
- Швидкими пропозиціями
- Контекстними відповідями
- Демо режимом

#### Інтерактивна мапа курорту
SVG мапа з:
- Маркерами локацій
- Hover ефектами
- Інформаційними картками
- Посиланнями на сторінки
